Setting Y-Axis Rounding

The y-axis values can be set to a rounding precision with the settings.axis.rounding attribute.

The default value for rounding is 2 decimal places.

A Small Example


chart: {
  data: [[200,400,500,600,400,400,300],[500,300,200,200,300,400,500]],
  settings: {
    draw: {
      axis: {
        rounding: 2
      }
    }
  }
}
