Setting Chart Padding

The padding of the chart can bet set with four different attributes: settings.padding.top, settings.padding.bottom, settings.padding.left, and settings.padding.right.

The default values for padding are as follows:

  1. top: 20
  2. bottom: 20
  3. left: 100
  4. right: 20

A Small Example


chart: {
  data: [[200,400,500,600,400,400,300],[500,300,200,200,300,400,500]],
  settings: {
    padding: {
      left: 200,
      right: 200,
      top: 100,
      bottom: 100
    }
  }
}
