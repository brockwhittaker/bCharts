Creating Line Color Options

The line colors of charts can be easily set through the settings.line.color attribute.

The default value for line colors is an array of values, rgb(100,169,197), rgb(230,106,124), rgb(151,110,201), rgb(169,201,110), rgb(230,176,99).

A Small Example


chart: {
  data: [[200,400,500,600,400,400,300],[500,300,200,200,300,400,500]],
  settings: {
    line: {
      color: ["#B45656", "rgb(80, 148, 134)"]
    }
  }
}
