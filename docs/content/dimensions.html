Setting Chart Dimensions

The dimensions of the chart can bet set with two different attributes: settings.dimensions.width, and settings.dimensions.height.

The default values for padding are as follows:

  1. width: 1000
  2. height: 600

Keep in mind however that these values can scale proportionally to any resolution.

A Small Example


chart: {
  data: [[200,400,500,600,400,400,300],[500,300,200,200,300,400,500]],
  settings: {
    dimensions: {
      width: 800
      height: 800
    }
  }
}
