Enabling and Disabling Y-Axis

The y-axis can be enabled and disabled with a boolean in the settings.draw.axis.y attribute.

The default value for Y-Axis values is true.

A Small Example


chart: {
  data: [[200,400,500,600,400,400,300],[500,300,200,200,300,400,500]],
  settings: {
    draw: {
      axis: {
        y: false
      }
    }
  }
}
