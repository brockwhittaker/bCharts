Enabling and Disabling Lines

The axis lines can be enabled and disabled with a boolean in the settings.draw.axis.lines attribute.

The default value for axis lines is true.

A Small Example


chart: {
  data: [[200,400,500,600,400,400,300],[500,300,200,200,300,400,500]],
  settings: {
    draw: {
      axis: {
        lines: false
      }
    }
  }
}
